cuicui_paintesting.sh:

	Script bash de base.	
	Permet de lancer une auto-compilateur qui se repete a intervale regulier.
	Permet de lancer de multiple test sur des program specifique.
	
	Les fichier sont a place a la racine du projet a tester.


	
run_make.sh:
	
	Execute en booucle une commande make afin de compiler le programe avec
	les changements effectuer dans le code.




clean.sh:
	se script permet de clean le logiciel de paintest




Le dossier paintest contient les different test destiner au program a tester
