# __Cube3D__

Cube3D est un projet de l'école 42 qui consiste a creer un jeu en 3D en utilisant la librairie mlx de l'école 42 et l'algorithme de raycasting.

> ce projet inclu la libmlx en vrsion linux et MacOS
